Game Basics

Code & Conquer is a turn-based strategy game where you control units by writing simple commands.

Command Types

  • move(steps, direction) - Move your unit
  • attack() - Attack adjacent enemies
  • wait() - Skip a turn

Examples

move(2, "right")

move(1, "up")

attack()

Drag & Drop Coding

You can drag commands from the panel into the code editor:

  1. Click and hold a command button
  2. Drag it to the code editor
  3. Release to insert the command

Game Objectives

Defeat Enemies

Use attack commands to eliminate enemy units

Reach Objectives

Move to specific tiles to complete levels

Survive

Avoid hazards and enemy attacks
